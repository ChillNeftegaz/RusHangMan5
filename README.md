# Русская виселица

Классическая игра «Виселица» на русском языке. Нужно угадать слово по буквам. Каждая ошибка приближает появление спрайта. Полный спрайт — поражение.

## Игровой процесс

- На экране отображается загаданное слово (в виде прочерков) и алфавит
- Игрок нажимает на буквы
- Если буква есть в слове, она открывается на своих позициях, кнопка становится зелёной и блокируется
- Если буквы нет в слове, кнопка становится красной и блокируется, а счётчик ошибок растёт
- При каждой ошибке справа появляется часть виселицы
- Когда спрайт полностью отрисован — игра заканчивается поражением
- Если слово угадано полностью — победа

## Особенности

- Слова загружаются случайным образом из встроенного словаря
- После окончания игры показывается загаданное слово
- Анимация пульсации при нажатии на букву
- Белая вспышка при перезапуске
- Кнопка «Начать заново» перезапускает игру без перезагрузки страницы

## Технологии

- HTML
- CSS (анимации, градиенты, грид-раскладка)
- JavaScript (DOM-манипуляции, обработка событий, управление игровым состоянием)

## Установка и запуск

1. Скачай или склонируй репозиторий
2. Открой файл `index.html` в любом браузере
3. Игра не требует интернета и работает локально
